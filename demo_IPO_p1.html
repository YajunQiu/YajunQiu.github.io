Project Description

A 50-minute elearn course built with Articulate Storyline in 2021. It is composed of 10 modules under a main menu, while the learner will be assigned a certificate after completing all modules which stands for 10 key steps in IPO process.

Project Goal

Aiming to build up an IPO focused learning package covering both technical and non-technical capabilities. The deliverables will be scenario-based and fit into the workflow, which support learning on demand.
