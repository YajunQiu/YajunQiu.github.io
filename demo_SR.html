Project Description

Salamander Rescue is 3D immersive game for 6th graders to learn science. Salamander Rescue Website is the official Website for the the game.

Project Goal

The primary goal is to create Salamander Rescue Website to bring more attention to an immersive 3D learning game---Salamander Rescue. It serves to introduce the content, key design features, as well as the direction for Salamander Rescue users.

My Role and Duties

  • Leading UI/UX designer, multimedia content developer and Web developer
  • Conducting user analysis, creating a user personae and user story, and then drafting web workflow and web frame.
  • Developing web content such as gameplay tutorial video by Camtasia and Premiere and game character headshot by Photoshop
  • Developing Website with code of HTML5 and CSS3

Tools Used

HTML5, CSS3, Gliffy, Axure, Adobe Photoshop, Adobe Premiere, Camtasia, FileZilla
