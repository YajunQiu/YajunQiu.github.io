Project Description

Barrel Up is a user-friendly mobile app that provides an all-in-one solution for brewers to manage barrels in a highly intuitive and efficient way. With the help of a QR scanner and cloud database, brewers can use the Barrel Up app to manage their barrel database on their mobile devices.

Project Goal

The primary goal is to design a mobile app based on brewers' needs so as to improve the efficiency of their workflow, increase the productivity of their barrel programs, and decrease instances of error, coming up with a solution that would allow them to both track and record their barrels and beers in the same database.

My Role and Duties

  • Project manager and leading UI/UX designer
  • Conducting user analysis and competitive evaluation, creating a user personae and user story, and then drafting web workflow and web frame.
  • Leading design team to design mobile app based on the user research
  • Developing Mockup with Axure
  • Conducting user testing both in person and online and revise the mockup accordingly.
