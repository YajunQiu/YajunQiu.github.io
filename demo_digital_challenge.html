Project Description

An mobile-friendly quiz on training result regarding Cybersecurity through interactive game environment developed in Articulate360.

Project Goal

As a key component in Digitisation Learning Journey, it is designed as a gamified and immersvie learning experience with fun that delivered at the end of the journey. Target learners are company staffs (from A1 to Partner) and testing result recorded for further analysis. The quiz is tailored to use both on desktop or mobile.
