Project Description

There is a growing need for amateurs to design their own website. Website Builders offer users a convenient way to create personalized websites by providing well designed templates. Design a usertesting to evaluate the usability of Wix and Weebly and provide suggestion for web revision.

Project Goal

The purpose of this study is to evaluate the usability of the website builders by comparing two market-leading tools , which are Wix(www.wix.com) and Weebly(www.weebly.com).

My Role and Duties

  • Leading user-testing designer
  • Creating usability test plan and testing materials.
  • Conducting user testing, playing a role as a recorder and moderator
  • Analyzing usertesting data and conduct data visualiztion
  • Documenting the usertesting process, wrtiting report and listing revision suggestion
